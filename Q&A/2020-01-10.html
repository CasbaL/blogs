# 集成 Travis CI 自动部署到 GitHub page

Travis Ci 官网(直接用 GitHub 授权登录)
GitHub pages help要使用 GitHub Pages 只需新建一个名字为 username.github.io 的仓库，然后把打包好的文件push 到这个仓库的 master 分支就行；

我们先打开这个链接 这里已经有很明细的教程；

  1. 先在项目根目录下创建 .travis.yml 文件； 可以直接copy官网的是示例；
    deploy:
  provider: pages
  skip_cleanup: true
  github_token: $GITHUB_TOKEN  # 这里的GITHUB_TOKEN是一个变量
  keep_history: true
  on:
    branch: master
